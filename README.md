# Portfolio

A modern personal portfolio for **Ikram Ulllah**, built with HTML, CSS, and JavaScript. The site highlights web development services, featured projects, and social links for the Hotak Script brand.

## Overview

This project is a static-first portfolio website focused on presentation and motion. It includes a bold hero section, animated skill cards, project highlights, and social media links.

## Features

- Responsive single-page portfolio layout
- Animated hero, reveal effects, tilt cards, and particle background
- Light and dark theme toggle with saved preference
- Featured projects section with live preview links
- SEO-friendly metadata, `robots.txt`, and `sitemap.xml`

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

1. Clone the repository.
2. Open the project folder.
3. Launch `index.html` in your browser or deploy it as a static site.
